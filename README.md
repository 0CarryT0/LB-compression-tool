# LB压缩工具 vol1

LB压缩工具基于算术编码对文件进行无损压缩。

### 工作方式

压缩：file_name ---> file_name.LB

解压：file_name.LB ---> file_name

### 使用方法

请确保有pyhton运行环境。

以下命令行操作仅供参考。

在工作文件夹中打开cmd，使用命令行如下：

```
python Arithmetic_main.py
```

然后根据指引输入模式、文件路径即可。
